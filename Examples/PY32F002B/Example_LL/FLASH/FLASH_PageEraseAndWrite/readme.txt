功能描述：
此样例演示了flash page擦除和page写功能。

Function descriptions:
This sample demonstrates the flash page erase and page write functions.

================================================================================
测试环境：
测试用板：PY32F002B_STK
MDK版本： 5.28
IAR版本： 9.20

Test environment:
Test board: PY32F002B_STK
MDK Version: 5.28
IAR Version: 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行；
2. 按下按键，程序进行page擦除和page写功能；
3. page擦除成功和page写成功，LED亮。

Example execution steps:
1. compile and download the program to the MCU and run it;
2. press the key, the program performs the function of page erase and page write;
3. page erase successfully and page write successfully, the LED on.
