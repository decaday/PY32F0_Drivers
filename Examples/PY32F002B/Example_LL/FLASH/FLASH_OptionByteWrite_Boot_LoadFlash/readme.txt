================================================================================
                                样例使用说明
                                Sample Description
================================================================================
功能描述：
此样例演示了修改启动模式从LoadFlash启动，并设置LoadFlash的大小为3k。

Function descriptions:
This sample demonstrates modifying the boot mode to boot from LoadFlash and setting
the size of LoadFlash to 3k.

================================================================================
测试环境：
测试用板：PY32F002B_STK
MDK版本： 5.28
IAR版本： 9.20

Test environment:
Test board: PY32F002B_STK
MDK Version: 5.28
IAR Version: 9.20
================================================================================
使用步骤：
1. 编译下载程序并运行
2. 按下按键
3. 程序修改启动模式从LoadFlash启动，并设置LoadFlash为3K

Example execution steps:
1. Compile and download the program and run it
2. Press the button
3. Modify the boot mode to start from LoadFlash, and set LoadFlash to 3K
